# 🎮 The Last Stand

## Description
2D top-down zombie survival game where a survivor fights zombies, collects resources, activates safe zones, and tries to reach extraction.

## 👥 Team
- Altynbek
- Olzhas

## 🛠️ Technologies
- Java
- libGDX

## 🧠 Design Patterns
- Factory Method – creating enemies and items
- Abstract Factory – generating different zones
- Decorator – effects like healing or damage boost
- Facade – managing game systems
- Singleton – global managers
- Composite – mission objectives

## 🎯 Features
- Common and special zombies
- Resource system (medkits, ammo)
- Safe zones (4 zones to activate)
- Basic survival gameplay
